Das Beispiel soll zeigen, wie ein WPF Dialog im SysTray verkleinert werden kann und vor dort mit einem Kontextmenü Fuktionen ausgeführt werden können.

Ein Symbol in der Taskleiste, auch als Benachrichtigungssymbol oder Taskleisten-Benachrichtigungssymbol bezeichnet, ist eine kompakte visuelle Darstellung, die im Benachrichtigungsbereich der Taskleiste oder der Systemablage eines Desktops angezeigt wird. Sein Hauptzweck besteht darin, einen bequemen Zugriff auf die Funktionen von Anwendungen zu ermöglichen oder dem Benutzer Benachrichtigungen anzuzeigen.
